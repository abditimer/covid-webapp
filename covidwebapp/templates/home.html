The Covid Data Project.

This website has been built with the intention to be a technical website. The analysis should not be considered scientific, it has been created to better understand the technologies that make up the individual components of this website. This includes python, flask and plotly. For a more scientific study, please refer to the orginal source. Our Data page goes into detail into the data, providing a variety of charts that visualise the current COVID 19 rates in England. Below, you can find an example chart.

Deaths in England split by Region.

The chart above shows the death count in England, split into the different regions in the UK. for example, you can see that both London and the midlands have a high death count, with the north west closely following. In the data page, we will look further into the data, as we try to get a clearer picture of the current state in the uk.
